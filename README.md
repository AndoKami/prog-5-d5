## exemple d'utilisation

```bash
=== Machine à Café Automatique ===
Bienvenue ! Voici les boissons disponibles :
1. Espresso - 2.5€
2. Americano - 3€
3. Cappuccino - 4€
4. Latte - 4.5€

Commandes disponibles :
- pay <montant> <méthode> (ex: pay 5 Cash)
- select <boisson> (ex: select Espresso)
- prepare
- maintenance
- reset (en cas d'erreur)
- exit

Entrez une commande : pay 5 Cash
Paiement de 5€ via Cash accepté.
Entrez une commande : select Espresso
Café Espresso sélectionné.
Entrez une commande : prepare
Préparation du Espresso en cours...
Distribution du café dans le gobelet.
Rendu de monnaie : 2.5€
Espresso prêt !
